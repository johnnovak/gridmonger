About ¶

Gridmonger is a free and open-source dungeon mapping tool to replace pencil-and-paper mapping when playing old-school computer role-playing games (cRPGs). It aims to do one thing and do it exceedingly well:

Be the ultimate keyboard-driven grid-based cRPG mapping companion!

Gridmonger is available for free for Windows and macOS (Linux version is in the works).

Most notable features ¶

  • (Mostly) keyboard-driven interface for maximum mapping efficiency (similar to Vim and Blender)
  • Elegant scalable user interface with high-DPI support
  • Extensive theming options & 33 included themes
  • Four different navigational & editing modes for the best map-making experience while playing the game
  • Create maps as large as 999 levels, up to 6,666 × 6,666 cells per level
  • Support for all common map styles (wall style, tunnel style, single big world map with regions)
  • Linked teleports, stairways, and doors for easy navigation
  • Various cell annotation options (e.g., numbers, icons, comments)
  • Notes list pane to view all annotations in one place and filter them by various criteria, including full-text search
  • Manipulate the whole map or a selection with handy tools (copy/paste, move, crop, resize, move/fill/erase area, etc.)
  • Wraparound option when navigating or manipulating the map
  • Diagonal movement support
  • Unlimited undo & redo history

Linux

No Linux packages are available yet, but you can try to build Gridmonger yourself by following the build instructions. There might be some graphical glitches when resizing and moving the application window, but otherwise the program works perfectly.

Example maps ¶

You can download the Example Maps included in the installer packages separately as well.

Note that some of these maps are incomplete and not necessarily 100% accurate; they are only meant to showcase the capabilities of the program. Besides, you wouldn't rob yourself of the fun of making your own maps, would you?
